Bingham High iOS App

The Bingham iOS App contains a suite of useful features for new and existing Bingham students. It is used by hundreds today.

Users are presented with this screen when launching the app. It contains a layout to access all of the features within the app.

This feature allows students to see their location within Bingham and quickly search for classrooms they need to find.

The GPA calculator allows students and coaches to quickly calculate their GPA.

The homework planner ensures students never forget to do their assignments. Students input their homework and they receive a push notification reminder before it's due.

Students can view the length of each class with the bell schedules feature.

WebDevTutorials

Nations 11O

Nations 11O was a class dashboard which students and teachers used to communicate announcements and events. Students also used it to quickly access various resources on the internet.

The home page of Nations 11o includes 1-click access to tons of resources. A large calendar displays upcoming events, the menu bar along the top has a dropdown with links to other websites, and the boxes on the left side contain announcements from teachers blogs.

As the user scrolls down, they see announcements which teachers and students can add to the dashboard, and links to more useful resources.

When a user clicks on one of the tiles, more information appears in a popup.

Students and teachers can add events to the dashboard by clicking on the add tile. A popup with a form to add the message appears.
